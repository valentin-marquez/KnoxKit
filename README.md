# KnoxKit

KnoxKit is a Project Zomboid instance manager and modpack tool — a ground-up Tauri 2 rewrite of the archived Electron-based KnoxKit v1. It lets players create and manage isolated game instances, assemble and sync modpacks, and launch Project Zomboid with per-instance configuration.

See `docs/dev-setup.md` for setup.

**Stack:** Tauri 2 + Rust backend, React 19 + TypeScript + Vite frontend, Bun toolchain, Biome for lint/format, `just` for task running, `mise` for toolchain pinning.
